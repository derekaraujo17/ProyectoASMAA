Proyecto enfocado en la investigación, diseño y realización de tu resumen mensual sobre tus artistas y canciones más escuchados; junto con las horas que reprodujiste música. (EN PROCESO). Spotify Wrapped Mensual

Spotify Wrapped Mensual es una aplicación que genera un resumen mensual personalizado de tu actividad musical, mostrando: *Top canciones del mes *Artistas más escuchados *Tiempo total de reproducción *Estadísticas y visualizaciones dinámicas

Inspirado en Spotify Wrapped, pero con una frecuencia mensual para ofrecer un análisis más constante y detallado de tus hábitos musicales.

El objetivo es desarrollar una aplicación que: +Permita al usuario conectar su cuenta de Spotify. +Obtenga datos mediante la API oficial de Spotify. +Procese la información mensual. +Genere estadísticas atractivas y visualmente amigables. +Muestre un resumen interactivo tipo “historia” o dashboard.

Tecnologías Utilizadas: (Aqui se pondran una vez este decidido todo).

El proyecto se desarrolla utilizando una metodología Kanban organizando el trabajo con checklists de: Por hacer, En proceso y Hecho. Retroalimentación en cada momento y abiertos a opinar y sugerir nuevas dinámicas.
